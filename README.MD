# Video Game Store

## Features
* Filter Categories
* Filter Subcategories
* Filter Price
* Add Items To Cart
* Update Quantity Of Items In Cart
* Clear Cart
* Checkout
* Register New Users
* Login
* Edit User Profile

## Includes
* Main Shop Screen
* User Profile Screen
* Unique User Cart Screen
* Checkout Screen - not added
